# Memory Templates for Second Brain

Quick-reference templates for adding memories. Copy and paste as needed.

---

## 1. Meeting Notes

```
## Meeting: [Meeting Title]

**Date:** YYYY-MM-DD
**Time:** HH:MM
**Attendees:** @Name1, @Name2, @Name3
**Company:** [[Company Name]]
**Tags:** #meeting #partner #opportunity

### Agenda
1. Topic 1
2. Topic 2
3. Topic 3

### Discussion
- Key point 1
- Key point 2
- Key point 3

### Decisions Made
- Decision 1
- Decision 2

### Action Items
- [ ] Action 1 - @Owner - Due: Date
- [ ] Action 2 - @Owner - Due: Date

### Next Steps
- Next meeting scheduled for: Date
- Follow-up required with: @Name
```

## Quick Reference

### Common Tags

| Tag | Use For |
|-----|---------|
| #opportunity | Sales opportunities |
| #partner | Partner discussions |
| #tender | RFP/tender responses |
| #demo | Demos and presentations |
| #follow-up | Follow-up tasks |
| #hot-lead | High priority opportunities |
| #poc | Proof of concept |
| #pricing | Pricing discussions |
| #technical | Technical discussions |
| #strategy | Strategic discussions |

### Formatting Tips

- **Companies:** `[[Company Name]]` - Creates wiki link
- **Contacts:** `@First Last` - Mentions contact
- **Tags:** `#tag-name` - Adds tag
- **Actions:** `- [ ] Task description` - Checkbox item
- **Priority:** `!!!` in action = High priority
- **Dates:** `YYYY-MM-DD` format for consistency
